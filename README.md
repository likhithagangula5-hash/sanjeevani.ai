# 🚑 SanjeevaniAI

## AI-Powered Ambulance Route Optimizer & SMS Alert System

### 📖 About the Project
SanjeevaniAI is an intelligent emergency healthcare solution designed to reduce ambulance response time and improve communication during medical emergencies.

The system helps identify the fastest route for ambulances and automatically sends SMS alerts to hospitals and emergency contacts, ensuring quicker emergency response and better patient care.

---

## 🎯 Objective

To improve emergency healthcare services by:
- Reducing ambulance travel time
- Optimizing routes using intelligent algorithms
- Providing instant SMS notifications
- Enhancing coordination between patients, hospitals, and emergency services

---

## ✨ Features

- 🚑 Ambulance Route Optimization
- 📩 Automated SMS Alert System
- 🏥 Hospital Notification
- 📍 Fastest Route Identification
- 👨‍⚕️ Emergency Response Support
- 📱 User-Friendly Interface

---

## 🛠 Technologies Used

---

## ⚙️ System Workflow

1. User requests emergency assistance.
2. System identifies the best available route.
3. Route optimization algorithm calculates the fastest path.
4. SMS alerts are sent to hospitals and emergency contacts.
5. Ambulance reaches the destination with reduced delay.
